This is a standalone packaging of the Unionfs filesystem for use with SUSE Linux Enterprise Server.  No patching of the kernel is required.

Please note that this is an unofficial packaging and retrofit, so don't blame the unionfs authors for any bugs I may have introduced.

Download SLES10 source:

unionfs-2.5.5-1.src.rpm

Download SLES10 SP3 binary package:

unionfs-kmp-default-2.5.5_2.6.16.60_0.66.1-1.s390x.rpm
