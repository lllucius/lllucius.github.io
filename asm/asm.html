ARC4 Encryption

An external Rexx function implementing the "Alleged" RC4 (or ARCFOUR/ARC4) algorithm.  This algorithm purports to provide the same encryption/decryption as the RC4 algorithm by RSA Security.

MD5 Message-Digest

An implementation of the MD5 Message-Digest as documented in RFC 1321.

Base64 Encoding

External Rexx function implementing the base64 encoding scheme documented in RFC 2045.

Ascii85 Encoding

External Rexx function implementing the ascii85 encoding scheme as used by the Postscript and PDF file formats.

zlib390 Compression Library

zlib compression library for IBM 9672-G3+, zSeries, and compatible mainframe.

zlibLE Compression Library

zlib compression library for IBM's Language Environment.

bzlib390 Compression Library

bzip2 compression library for IBM 9672-G3+, zSeries, and compatible mainframe.

bzlibLE Compression Library

bzip2 compression library for IBM's Language Environment.

HETUTL Tape Utility

Tape conversion utility for use with the Hercules mainframe emulator.
