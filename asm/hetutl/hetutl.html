HETUTL is a utility that is run on IBM's OS/390+ operating system to convert physical tapes to disk files in the HET format.  These HET files can then be used by the Hercules mainframe emulator to simulate the real tape.

The program accepts an IBM format tape and converts it to an HET format file. Output compression, input block sizes of up to 65535 bytes, and options to create files compatible with the AWSTAPE format are supported.

It uses the zlib390, bzlib390, zlibLE, and/or bzlibLE libraries that can be found here for its compression routines.
